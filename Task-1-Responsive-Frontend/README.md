# Task 1 — Responsive Frontend Interface

A personal developer portfolio built with **pure HTML5, CSS3, and vanilla JavaScript** — no frameworks.

## What this demonstrates
- **Responsive design**: mobile-first layout using CSS Grid + Flexbox, with breakpoints at 768px (tablet) and 1024px (desktop)
- **Fluid typography** via `clamp()` instead of fixed breakpoint-only font sizes
- **Semantic HTML5** landmarks (`header`, `nav`, `main`, `section`, `footer`) for accessibility and SEO
- **Dark/light theme toggle** with `localStorage` persistence
- **Interactive JavaScript**: mobile hamburger menu, typing animation, scroll-reveal animations, client-side form validation
- **WCAG-conscious accessibility**: skip link, ARIA labels, focus-visible states, reduced-motion support

## Files
- `index.html` — markup
- `index_Style.css` — all styling
- `index_Script.js` — all interactivity (note: this version also includes the Task 4 backend integration — see that folder for details)
- `Project1_Discovery_Document.docx` — research, empathy map, and "How Might We" problem framing (Pillar 1 of the brief)
- `Project1_Wireframe_Document.docx` — low-fidelity grayscale wireframe completed before any styling decisions were made

## How to view
Open `index.html` directly in a browser, or serve the folder with any static file server (e.g. `python -m http.server`).
